# Traffic Density Detection Using OpenCV

This project detects vehicles in real time using OpenCV and Haar Cascade.
It counts the number of cars and determines traffic density.

## Technologies
- Python
- OpenCV
- Haar Cascade
- webcam

##Features
- Detect vehicles from live camera feed
- Count number of cars
- Analyze traffic density
- Display traffic status

## Run
pip install opencv-python imutils
python traffic_detection.py
